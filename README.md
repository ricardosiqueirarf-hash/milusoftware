# MILU Software

Documentação pública oficial do **MILU Software**, uma plataforma de gestão e orçamento voltada a fabricantes e empresas que trabalham com produtos personalizados de alumínio, vidro e marcenaria.

Este repositório contém somente informações públicas: visão geral do produto, primeiros passos, módulos, integrações, segurança, perguntas frequentes e preparação para APIs públicas.

> O código-fonte, a arquitetura interna, as estruturas de banco de dados, as credenciais e as regras proprietárias do MILU não fazem parte deste repositório.
