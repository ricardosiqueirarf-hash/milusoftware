# Contribuindo com a documentação pública

Este repositório aceita contribuições relacionadas exclusivamente à documentação pública do MILU Software.

## Contribuições adequadas

- correções de ortografia e clareza;
- melhorias de navegação;
- exemplos públicos sem credenciais;
- dúvidas frequentes;
- correções de links;
- sugestões de estrutura da documentação.

## Não publique

- código-fonte privado;
- regras proprietárias de clientes;
- credenciais, tokens ou certificados;
- dados pessoais;
- documentos fiscais ou bancários;
- detalhes exploráveis de segurança;
- estruturas internas completas do sistema.

## Fluxo sugerido

1. Crie uma branch descritiva.
2. Altere somente os arquivos relacionados à proposta.
3. Revise links, linguagem e exposição de informações.
4. Abra um pull request explicando o que mudou e por quê.

Contribuições podem ser ajustadas ou recusadas para preservar consistência, segurança e posicionamento público do produto.
